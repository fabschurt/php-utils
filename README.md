# Php\Utils

This is a collection of highly focused utility classes to use in your PHP
projects.

## Requirements

* [PHP](https://secure.php.net/) ~7.0

## Installation

Just require `fabschurt/php-utils:@dev` with [Composer](https://getcomposer.org/)
and you're good to go.
